what is string

In JavaScript, a string is a primitive data type that represents a sequence of characters. Strings are used to store and manipulate text in a computer program. A string in JavaScript is created by enclosing a sequence of characters within single quotes (') or double quotes (").

Why use stirng :

Strings in JavaScript are used to represent and manipulate text. They are a fundamental data type in the language and serve various purposes in web development and programming in general.

Explain the difference between single quotes ('), double quotes ("), and backticks (`) when defining strings.

Single Quotes ('): Traditional way to define strings. No expression interpolation or multiline support. Escape characters needed for nested single quotes.
Double Quotes ("): Similar to single quotes. No expression interpolation or multiline support. Escape characters needed for nested double quotes.
Backticks (`): Introduced in ES6. Support multiline strings and expression interpolation using ${}. Preferred for more dynamic and complex strings.
