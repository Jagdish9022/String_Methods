Upper and Lower case

toUpperCase(): The toUpperCase() method transforms all characters in a string to uppercase. This means that if a string contains a mix of uppercase and lowercase letters, calling toUpperCase() on that string will convert all the letters to uppercase.

toLowerCase(): Conversely, the toLowerCase() method converts all characters in a string to lowercase. If a string contains a mix of uppercase and lowercase letters, calling toLowerCase() on that string will transform all the letters to lowercase.
